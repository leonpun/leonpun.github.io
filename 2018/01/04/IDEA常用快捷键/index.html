IntelliJ IDEA笔记

发表于 2018-01-04

本文之前是记录查过的IDEA快捷键，最近使用小有心得，系统整理一下。

使用频度最高的：alter + enter、Ctrl + Alt + A Find Action

高效定位代码

  • 项目间跳转

Ctrl + Alt + [ or ] 切换项目

  • 文件间跳转

    Ctrl + E 最近的打开的文件
    Ctrl + Shift + E 最近修改的文件
    Alt+ left/right 切换代码视图

  • 浏览修改位置的跳转

    Ctrl + Shift + Backspace 上次修改的地方
    Ctrl + Alt + left/right 返回至上次浏览的位置

  • 无处不在的跳转

    F11 添加书签
    Ctrl + F11 编号书签
    Ctrl + 上述编号 跳转到书签
    Alt + Shift + F 收藏
    Alt + 2 打开收藏工具栏
    编辑区和项目区跳转
    Alt+ Up/Down 在方法间快速移动定位

  • 精准搜索

    Ctrl + N 搜索类
    Ctrl + Shift + N 搜索文件
    Ctrl + Shift + Alt + N 搜索符号（方法、变量）

代码小助手

  • Live Template

  • postfix

    return

编写高质量代码

  • 重构

    Shift + F6 重命名
    Ctrl + F6 修改方法签名

  • 抽取

    Ctrl + Alt + V 抽取变量
    Ctrl + Alt + C 抽取常量
    Ctrl + Alt + F 抽取成员变量
    Ctrl + Alt + P 抽取方法参数
    Ctrl + Alt + M 抽取函数

    Ctrl + F12 弹出当前文件结构层
